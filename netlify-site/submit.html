How It Works

Fill out the form below. Click submit to open your email with the details pre-filled. The TechNote team will review and add your app if approved.

Submission Guidelines

Requirements

  1. App must be a single .py file
  2. Must inherit from SoftApp (TechNote app framework)
  3. Must use self.speak() for all voice output
  4. Must handle keyboard input via on_key() and on_key_up()
  5. No external dependencies beyond Python standard library
  6. No harmful, malicious, or inappropriate content

Review Process

  1. You submit via email
  2. Team reviews your code
  3. If approved, your app is added to the catalog
  4. You get notified when it goes live
