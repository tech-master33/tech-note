Tic Tac Toe

Games Version 1.0 • Single Player vs Computer

About

The classic game of Xs and Os. Play against the computer on a 3x3 grid. Take turns placing your mark — first to get three in a row wins. Features a smart computer opponent that blocks your moves and sets up traps.

Preview

X | O | X --------- | X | --------- O | | X You: 3 Computer: 2 Ties: 1

Controls

  • Arrow keys — Move cursor to select cell
  • Enter — Place your mark (X)
  • Escape — Exit game

Features

  • Smart AI opponent
  • Score tracking across rounds
  • Instant win/draw detection
  • Voice announcements

How to Install

Open TechNote → App Store → Tic Tac Toe → Press Enter to install. It will appear in your Game Center.
