Typing Test

Apps Version 1.0 • Productivity

About

Measure your typing speed in words per minute (WPM) and accuracy. Choose from 30, 60, or 120 second tests. Type the displayed text as fast as you can — results are announced when time runs out.

Preview

Time: 24s Text: the quick brown fox jumps over lazy dog python keyboard screen window folder mouse You: the quick brown fox _

Controls

  • Letter keys — Type characters
  • Space/Enter — Word break
  • Backspace — Delete last character
  • Up/Down — Select duration
  • Escape — Cancel / exit

Features

  • 30, 60, or 120 second tests
  • WPM and accuracy tracking
  • 60+ word bank
  • Real-time feedback

How to Install

Open TechNote → App Store → Typing Test → Press Enter to install.
