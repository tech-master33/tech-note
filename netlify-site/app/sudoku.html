Sudoku

Games Version 1.0 • Number Puzzle

About

The classic 9x9 number puzzle. Fill every row, column, and 3x3 box with the digits 1 through 9 — no repeats allowed. Choose from three difficulty levels that control how many cells are pre-filled. Tracks your errors so you can improve.

Controls

  • Arrow keys — Move cursor
  • 1-9 — Select digit to place
  • Enter — Place digit in cell
  • 0 — Clear cell
  • D — Cycle difficulty (Easy / Medium / Hard)
  • N — New game
  • Escape — Exit

Features

  • 3 difficulty levels
  • Proper puzzle generation
  • Error counter
  • Given cells are protected
  • Voice reads coordinates and values

How to Install

Open TechNote → App Store → Sudoku → Press Enter to install.
