Book Reader

Apps Version 1.0 • Read Text Files

About

Read text files from your TechNote documents folder. Navigate by page or line, set bookmarks, and use auto-read for hands-free reading. Supports adjustable reading speed.

Preview

Chapter 1: The Beginning It was a dark and stormy night when the adventure first began. The wind howled through the trees... Page 1 of 12 | Line 1-20

Controls

  • Space — Next page / line
  • Backspace — Previous page / line
  • Enter — Select / Play auto-read
  • B — Toggle bookmark
  • Escape — Exit

Features

  • Page-by-page or line-by-line navigation
  • Bookmark support for saving your place
  • Auto-read with adjustable speed
  • Reads .txt files from ~/.tech-soft/books/

How to Install

Open TechNote → App Store → Book Reader → Press Enter to install.
