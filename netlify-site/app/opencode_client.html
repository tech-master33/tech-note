Description

Full-featured AI client with support for 27 providers including OpenAI, Anthropic, Google, DeepSeek, and more. Manage sessions, switch models, and get voice output for all responses.

Features

  • 27 providers (OpenAI, Anthropic, Google, DeepSeek, Groq, OpenRouter, etc.)
  • Session management with chat history
  • Model selection per provider
  • Server, Cloud, and Direct connection modes
  • Agent switching (Build/Plan)
  • Voice output for all responses
  • Export chat history

Controls

  • Space Next item
  • Backspace Previous item
  • Enter Select
  • Escape Back / Exit
  • Any key Type message in chat

Installation

  1. Open TechNote and go to App Store
  2. Search for "OpenCode AI Client" or browse the Productivity category
  3. Press Enter to view details, then Enter again to install

Requires: Python 3.x, internet connection, API key for chosen provider
