Description

ChatGPT client with OpenAI and Zen API support. Manage chat sessions, select models, and get voice output for all responses.

Features

  • OpenAI API support (GPT-5.5, GPT-5.4, GPT-5.3, GPT-5.2, GPT-5.1, GPT-5)
  • OpenCode Zen API support (GPT-5.5, Claude Opus 4.8, Gemini 3.5, DeepSeek V4, MiMo-V2.5)
  • Session management with chat history
  • Model selection per provider
  • Voice output for all responses
  • Export chat history
  • Customizable system prompt

Controls

  • Space Next item
  • Backspace Previous item
  • Enter Select
  • Escape Back / Exit
  • Any key Type message in chat

Installation

  1. Open TechNote and go to App Store
  2. Search for "ChatGPT Client" or browse the Productivity category
  3. Press Enter to view details, then Enter again to install

Requires: Python 3.x, internet connection, OpenAI or Zen API key
