Connect Four

Games Version 1.0 • Single Player vs Computer

About

Drop your discs into a 7-column grid and try to connect four in a row before the computer does. Play vertically, horizontally, or diagonally on a 6x7 board. The computer uses smart strategy to block and attack.

Controls

  • Left / Right — Select column
  • Enter — Drop disc
  • Escape — Exit game

Features

  • 6x7 game board
  • Smart AI opponent
  • Win/loss tracking
  • Horizontal, vertical, and diagonal detection
  • Voice announcements

How to Install

Open TechNote → App Store → Connect Four → Press Enter to install.
