Calculator

Basic calculator with parentheses, history, and full voice feedback. Type numbers and operators, press Enter to calculate.

Calc: 3 + 5 * (2 - 1) = 8 History: 12 * 4 = 48 100 / 5 = 20 3 + 5 * (2 - 1) = 8

Controls

  • 0-9 - Type numbers
  • + - * / - Operators
  • ( ) - Parentheses for grouping
  • . - Decimal point
  • Enter - Calculate result
  • Backspace - Delete last character
  • Delete - Clear expression
  • Escape - Exit calculator

Features

  • Full voice feedback - speaks each character as typed
  • Says "plus", "minus", "times", "divided by" for operators
  • Parentheses support for complex expressions
  • Calculation history saved to disk
  • View history from menu
  • Clear entry or clear all
