Solitaire

Games Version 1.1 • Classic Card Game

About

The classic Klondike solitaire card game. Build four foundation piles from Ace to King by suit. Move cards between seven tableau columns, revealing hidden cards as you go. Draw from the stock pile when you run out of moves.

Preview

Stock: 5♥ Waste: J♠ Foundation: 3♦ 7♣ [K♠] Q♥ 8♦ 5♣ 2♠ J♦ 3♥ J♥ [7♠] 10♣ 4♦ 6♥ 1 2 3 4 5 6 7

Controls

  • Space — Next column
  • Backspace — Previous column
  • Enter — Select card / place card
  • F — Send card to foundation
  • N — New game
  • Escape — Exit

Features

  • Full Klondike solitaire rules
  • 7 tableau columns with face-down cards
  • Stock and waste pile
  • 4 foundation piles
  • Win counter
  • Voice describes every card

How to Install

Open TechNote → App Store → Solitaire → Press Enter to install.
