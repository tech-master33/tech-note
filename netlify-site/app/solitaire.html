Solitaire

Version 1.0 • Category: Games

Description

Classic Klondike solitaire. Build four foundation piles from Ace to King by suit. Move cards between tableau columns to reveal hidden cards.

How to Play

  • Arrow keys — Move cursor
  • Enter — Select or place card
  • F — Send card to foundation
  • Escape — Cancel or exit

Install

Open TechNote → App Store → Solitaire → Enter to install.
