2048

Version 1.0 • Category: Games

Description

Slide numbered tiles on a 4x4 grid. When two tiles with the same number touch, they merge into one. Reach 2048 to win.

How to Play

  • Arrow keys — Slide tiles
  • N — New game
  • Escape — Exit

Install

Open TechNote → App Store → 2048 → Enter to install.
