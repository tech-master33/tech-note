2048

Games Version 1.0 • Sliding Tile Puzzle

About

Slide numbered tiles on a 4x4 grid. When two tiles with the same number touch, they merge into one with double the value. Keep merging to reach the legendary 2048 tile. A new tile appears after every move — plan carefully or run out of space.

Preview

Controls

  • Arrow keys — Slide all tiles in that direction
  • N — New game
  • Escape — Exit

Features

  • Score and best score tracking
  • Continue playing after reaching 2048
  • Game over detection when no moves remain
  • Voice announces score after each move

How to Install

Open TechNote → App Store → 2048 → Press Enter to install.
