Minesweeper

Version 1.0 • Category: Games

Description

Reveal cells on a grid without detonating hidden mines. Use number clues to figure out where mines are and flag them.

How to Play

  • Arrow keys — Move cursor
  • Enter — Reveal or flag
  • Tab — Toggle reveal/flag mode
  • Escape — Exit

Install

Open TechNote → App Store → Minesweeper → Enter to install.
