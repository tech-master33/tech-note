Minesweeper

Games Version 1.0 • Logic Puzzle

About

Reveal cells on a 10x10 grid without detonating any of the 15 hidden mines. Each number tells you how many mines are adjacent. Use logic to flag mines and clear safe cells. First click is always safe.

Controls

  • Arrow keys — Move cursor
  • Enter — Reveal cell or place flag (depends on mode)
  • Tab — Toggle between Reveal and Flag mode
  • Escape — Exit

Features

  • 10x10 grid with 15 mines
  • Reveal and Flag modes
  • Auto-reveal empty cells
  • First click is always safe
  • Win/lose detection
  • Voice announces cell values

How to Install

Open TechNote → App Store → Minesweeper → Press Enter to install.
