Habit Tracker

Version 1.0 • Category: Apps

Description

Track daily habits and build streaks. Mark habits as done each day, view streaks, and check your stats. Data saved locally.

Controls

  • Space — Next item
  • Backspace — Previous item
  • Enter — Select
  • Escape — Exit

Open TechNote → App Store → Habit Tracker → Enter to install.
