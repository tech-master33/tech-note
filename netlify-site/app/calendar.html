Calendar

Apps Version 1.0 • Productivity

About

Full calendar with month view, day navigation, and event management. Add notes to any day — events are saved locally and persist between sessions. Today is highlighted with an asterisk.

Preview

Controls

  • Arrow keys — Navigate days (±1 day or ±7 days)
  • PageUp / PageDown — Change month
  • Enter — Open day options (add/delete events)
  • Escape — Exit

Features

  • Full month view with visual grid
  • Today highlighted automatically
  • Add and delete events per day
  • Events saved to disk permanently
  • Voice reads date and events

How to Install

Open TechNote → App Store → Calendar → Press Enter to install.
