Dice Roller

Apps Version 1.1 • Utilities

About

Roll multiple dice with modifiers for DnD, board games, or any tabletop game. Supports all standard dice sizes from d4 to d100. Visual ASCII dice faces show each roll. Roll history tracks your last 20 rolls.

Controls

  • Space — Next item
  • Backspace — Previous item
  • Enter — Select
  • Escape — Exit

Features

  • 1 to 20 dice per roll
  • All standard sizes: d4, d6, d8, d10, d12, d20, d100
  • Add or subtract modifier
  • Visual ASCII dice display
  • Roll history (last 20)
  • Voice announces total

How to Install

Open TechNote → App Store → Dice Roller → Press Enter to install.
