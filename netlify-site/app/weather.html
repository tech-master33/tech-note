Weather

Apps Version 1.1 • Utilities

About

Get current weather conditions for any city in the world. Shows temperature, humidity, wind, visibility, and UV index. Uses the free wttr.in service — no API key required.

Preview

Weather for London Temperature: 18C / 64F Feels like: 16C / 61F Conditions: Partly cloudy Humidity: 72% Wind: 15 km/h NW Visibility: 10 km UV Index: 5

Controls

  • Space — Next item
  • Backspace — Previous item
  • Enter — Select
  • Escape — Exit

Features

  • Works with any city worldwide
  • Temperature in Celsius and Fahrenheit
  • Wind speed and direction
  • UV index and visibility
  • Remembers last search

How to Install

Open TechNote → App Store → Weather → Press Enter to install.
