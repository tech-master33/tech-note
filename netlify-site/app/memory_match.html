Memory Match

Games Version 1.1 • Card Matching Puzzle

About

Test your memory by finding matching pairs of cards on a 4x4 grid. Flip two cards at a time — if they match, they stay revealed. Match all 8 pairs to win. Track your best time and moves.

Controls

  • Space — Next cell
  • Backspace — Previous cell
  • Enter — Flip card
  • N — New game
  • Escape — Exit game

Features

  • 4x4 grid with 8 matching pairs
  • Move counter
  • Win tracking
  • Voice announces flipped cards

How to Install

Open TechNote → App Store → Memory Match → Press Enter to install.
